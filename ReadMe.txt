The Game is started by simply executing the main class
Controls are the left and right arrow keys to naviagate around a room, and clicking a object in the game to use it.
EG a clicking a door will move you to the next room.
Note: You will need to resize the game window to actually play the game, and click on the graphics pannel at least once.

The game is implemented in a MVC patern, Where the world class is the controler, which calls model commands in the model package
Then Updates the view in the View package.
